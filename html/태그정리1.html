코생아 도현이의 코딩여행

도현이의 태그사전

  • li태그
    1. li는 자식 태그이다.

  • ul태그
    1. ul은 unordered list의 약자이다.
    2. ul은 부모태그이다.
    3. ul을 부모태그로 사용하면 그 속에 있는 li자식태그는 자동으로 ● 로 리스트화 한다.

  • ol태그
    1. ol은 ordered list의 약자이다.
    2. ol은 부모태그이다.
    3. ol을 부모태그로 사용하면 그 속에 있는 li자식태그는 자동으로 숫자. 으로 리스트화 한다.

  • strong, b태그
    1. strong 태그는 잘 안쓴다.
    2. b태그는 그나마 더 쓴다.

  • br 줄바꿈 태그
    1. / 태그가 필요없다.
    2. 강제로 한줄 띄우는 태그

  • h1~6 제목 태그
    1. 숫자가 커질수록 폰트 사이즈는 작아짐.
    2. 자동으로 볼드가 들어감.
    3. 자동으로 줄바꿈 들어감.

  • p 단락 태그
    1. 문단을 나누고 싶을 때 사용.
    2. / 태그 뒤로 줄바꿈이 들어감.
    3. br난사보다 p태그 사용할것.

  • img 이미지삽입 태그
    1. img 태그는 이름자체만으론 효력 발생x
    2. 태그의 속성을 활용해야함.
    3. 각 속성들의 순서는 상관이 없음.
    4. ex) img src=태그복습.png width=50%;

  • a 링크 태그
    1. hiper link 의 줄임말 h, ref = reference 는 참조라는 뜻
    2. 사용법 -> [a href="블라블라 링크" 하이퍼링크 걸려는 단어나 문장] [/a]
    3. ex) img src=태그복습.png width=50%;
    4. title="아무거나" -> 내가 링크에 마우스 포인터를 갖다대면 아무거나 라고 설명글이 나오게 하시오.
    5. target="_brank" -> 내가 링크를 눌렀을 때 새창으로 열어주시오.
