코생아 도현이의 코딩여행

css

  • 웹페이지에 CSS를 넣는 방법으로

    1) style태그를 사용하는 방법과(선택자=selector)
    2) style속성을 쓰는 방법이 있다.(효과=Declaration)

    • 선택자(Selector)와 효과(Declaration=선언)

  • 선택자(selector)는 CSS효과를 적용해 줄 대상을 가리켜 선택해주는 것이고, 선택한 태그 { 속성 } 과 같은 모양새 이다.
    head 태그 안에서 적용시키고 선택한 태그가 사용된 모든 내용에 {속성} 을 적용시킴.

  • 효과(declaration)는 개별 문장에 따로 style="속성" 을 사용해서 css를 넣어주는 방법.

  • {속성}은 적용되어 나타나게 될 CSS효과를 CSS문법을 통해 표현한 것으로, color:색상; text-decoration:원하는 효과; 등과 같이 생겼다.
    그리고 여러 개의 효과를 써준 경우, 세미콜론( ; )으로 각 효과를 구분해준다.
